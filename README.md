Aplicativo React Native com Expo que consome a API DummyJSON.

API utilizada: https://dummyjson.com/docs/products

- Listagem de produtos com `GET https://dummyjson.com/products`
- Tela de detalhes com `GET https://dummyjson.com/products/:id`
- Estados de carregamento, erro e recarregamento da lista
- Navegacao entre listagem e detalhes com React Navigation

1. Instale o Node.js LTS.
2. Instale as dependencias:

```bash
npm install
```

3. Inicie o projeto:

```bash
npm start
```

4. Abra pelo Expo Go no celular, emulador Android/iOS ou navegador:

```bash
npm run web
```

Para gerar a versao web:

```bash
npm run build:web
```

Projeto hospedado: https://paauloprado.github.io/dummyjson-api/
